# Tarea 2 - Optimización numérica en una dimensión

## Organización del proyecto

Todo el desarrollo de la tarea fue realizado dentro de un único **Jupyter Notebook**, donde se encuentra:

- Las implementaciones de los algoritmos.
- Los experimentos realizados.
- Las tablas de resultados.
- Los gráficos generados.

Por este motivo, el notebook constituye tanto el código fuente del trabajo como el soporte de los resultados presentados en el informe.

## Visualización del trabajo

El código, los resultados obtenidos y las figuras generadas pueden visualizarse directamente desde GitHub accediendo al notebook incluido en este repositorio.

De esta forma, es posible revisar tanto las implementaciones como las salidas obtenidas sin necesidad de ejecutar el proyecto localmente.

## Reproducción de resultados

En caso de querer verificar el correcto funcionamiento de todas las implementaciones y reproducir los resultados obtenidos, se recomienda:

1. Clonar el repositorio:

```bash
git clone https://github.com/belentellechea/obligatorio_2_calculo.git
```

2. Acceder al directorio del proyecto:

```bash
cd obligatorio_2_calculo
```

3. Abrir el notebook utilizando cualquier entorno compatible, como:
   - Jupyter Notebook
   - JupyterLab
   - Visual Studio Code
   
4. Ejecutar las celdas en orden de aparición.

Al ejecutar el notebook completo se regenerarán automáticamente todas las tablas, gráficos y resultados presentados en el informe.
